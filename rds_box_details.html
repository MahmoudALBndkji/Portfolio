RDS Box Details

Project information

  • Category :  Mobile Application
  • Project date :  7 Mar, 2024 ➡️ 2 Aug, 2024
  • Description :  The app lets customers receive online store deliveries anywhere, Users order from sites like Noon Store, specifying their location and a nearby storage box.
    Drivers deliver to the box, placing orders inside for secure pickup.
    This method ensures deliveries without customers needing to be present, adds protection through customer-guarded boxes, and offers convenience for travelers or those with delivery issues.
